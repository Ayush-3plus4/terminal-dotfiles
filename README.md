# ⚙️ Automated Dotfiles Manager

A lightweight bash script designed to instantly back up custom terminal configurations, aliases, and themes. 

## Features
* **Automated Backups:** Instantly copies core configuration files (like `.bashrc`) into a safe, version-controlled directory.
* **Disaster Recovery:** Ensures custom environments and personalized layouts are never lost if the system needs to be wiped or moved to a new machine.
* **Lightweight:** Pure bash implementation with zero external dependencies.
